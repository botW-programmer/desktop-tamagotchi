# 🍃 Desktop Tamagotchi Widget 

A cute, transparent desktop pet that passively monitors your computer's system resources! 

Unlike a normal Tamagotchi that gets hungry for food, this little creature gets stressed out when your PC is working too hard. Keep an eye on your RAM and CPU usage while adding a cozy, pixel-art aesthetic to your workspace.

## Features
* **Passive Monitoring:** Floats on your desktop without getting in the way of your active windows.
* **Live System Stats:** Calculates and displays real-time CPU and RAM percentage.
* **Dynamic Expressions:** Changes kaomoji expressions and colors based on how hard your PC is working!
* **Interactive Petting:** Click the creature to give it some attention and trigger a happy animation.
* **Fully Draggable:** Click and drag anywhere on the widget to move it around your screen.

## Built With
* **JavaScript (Node.js)**
* **Electron** (for window management and IPC bridging)
* **HTML/CSS** ## 🎮 How to Download and Run (No Coding Required!)
If you just want the widget on your desktop:
1. Go to the **[Releases](../../releases)** tab on the right side of this page.
2. Download the latest `.zip` file.
3. Extract the folder, double-click the `Desktop Tamagotchi.exe` file, and enjoy!

## Developer Setup
If you want to download the code and run it in development mode:
```bash
# Clone this repository
git clone [https://github.com/YOUR_USERNAME/desktop-tamagotchi.git](https://github.com/YOUR_USERNAME/desktop-tamagotchi.git)

# Go into the repository
cd desktop-tamagotchi

# Install dependencies
npm install

# Run the app
npm start
